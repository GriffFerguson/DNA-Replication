Help

Goal: Become polymerase and create the complementary strand of DNA for both the leading and lagging strand before times runs out.
To do this, select the correct nitrogenous base (adenine, thymine, cytosine, or guanine) for the selected blank from the menu on the left. The game will automatically select a different incomplete base pair, alternating between leading (5' to 3') and lagging (3' to 5') strands, when you complete a base pair.

Remember: The correct answer is the base that compliments the base of the original strand.

  • Adenine compliments thymine
  • Thymine compliments adenine
  • Cytosine compliments guanine
  • Guanine compliments cytosine
